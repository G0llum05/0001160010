Math.random()

Genera un numero decimale casuale tra 0 (incluso) e 1 (escluso):

console.log(random); // es: 0.7234891234

// Numero intero casuale tra 1 e 6

// Spiegazione:
// Math.random() ‚Üí 0.000 a 0.999
// * 6 ‚Üí 0.000 a 5.999
// Math.floor() ‚Üí 0 a 5
// + 1 ‚Üí 1 a 6
