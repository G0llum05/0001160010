Flexbox (Flexible Box Layout) √® un modulo CSS per creare layout flessibili e adattivi. Permette di posizionare elementi in modo fluido all'interno di un contenitore, adattandosi alla dimensione del contenitore e del dispositivo.

Concetti Base

  • Flex Container: L'elemento genitore con display: flex
  • Flex Items: Gli elementi figli diretti del container
  • Main Axis: L'asse principale (orizzontale di default)
  • Cross Axis: L'asse trasversale (verticale di default)

Propriet√† del Container

/* Direzione degli item */

/* Allineamento sull'asse principale */

Propriet√† degli Items

/* Riduzione proporzionale */
flex-shrink: <numero>; /* default: 1 */

/* Dimensione di base */

/* Shorthand */
