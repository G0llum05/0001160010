Struttura delle Tabelle

Le tabelle HTML sono utilizzate per organizzare e visualizzare dati in forma tabulare. Componenti principali:

  • <table>: Contenitore principale della tabella
  • <thead>: Raggruppa le righe di intestazione
  • <tbody>: Raggruppa il corpo della tabella con i dati
  • <tfoot>: Raggruppa il footer della tabella
  • <tr>: Table row - definisce una riga
  • <th>: Table header - cella di intestazione
  • <td>: Table data - cella di dati

Attributi Essenziali

  • colspan: Specifica quante colonne occupa una cella (th o td)
  • rowspan: Specifica quante righe occupa una cella (th o td)
  • <caption>: Aggiunge un titolo descrittivo alla tabella
  • <colgroup> e <col>: Definiscono propriet√† delle colonne
